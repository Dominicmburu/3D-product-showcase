Application Information
Project Details

This 3D App showcases three beverage product models with interactive controls. The application is built using Three.js for 3D rendering and Bootstrap for the user interface.

Included 3D Models
  • Jar Container - A brown container with ridged cap for easy opening
  • Soda Can - Bright red aluminum can with pull tab
  • Thermal Bottle - Yellow insulated bottle with metallic cap and handle
Interactive Features
  • Model selection with tabbed interface
  • Wireframe mode toggle
  • Auto-rotation toggle
  • Camera controls (rotate, pan, zoom)
  • Lighting controls (ambient, directional, spotlight)
  • Fullscreen mode
  • Responsive design for all screen sizes
  • Content swapping via tabbed interface
Technical Implementation
Technologies Used
  • HTML5 & CSS3 - Modern markup and styling
  • JavaScript (ES6) - Core programming language
  • Three.js - 3D rendering library
  • Bootstrap 5 - Responsive UI framework
  • Font Awesome - Icon library
  • Blender - 3D modeling software
