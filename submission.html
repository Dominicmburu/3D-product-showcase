Application Information
Project Details

This 3D App showcases three Coca-Cola brand beverage container models with interactive controls. The application is built using Three.js for 3D rendering and Bootstrap for the user interface.

Included 3D Models
  • Coke Can - Classic red Coca-Cola can design with iconic branding
  • Fanta Can - Bright orange container with vibrant Fanta branding
  • Sprite Bottle - Green-tinted transparent bottle with contoured design and Sprite logo
Interactive Features
  • Model selection with tabbed interface
  • Wireframe mode toggle
  • Auto-rotation toggle
  • Camera controls (rotate, pan, zoom)
  • Lighting controls (ambient, directional, spotlight)
  • Fullscreen mode
  • Responsive design for all screen sizes
  • Content swapping via tabbed interface
Technical Implementation
Technologies Used
  • HTML5 & CSS3 - Modern markup and styling
  • JavaScript (ES6) - Core programming language
  • Three.js - 3D rendering library
  • Bootstrap 5 - Responsive UI framework
  • Font Awesome - Icon library
  • Blender - 3D modeling software
Applied Assignment Concepts
  • Integration of 3D models with a responsive web interface
  • Implementation of wireframe mode toggle as required
  • Content swapping functionality for product information and media
  • Interactive lighting controls
  • Custom CSS logo and branding
  • Three Coca-Cola product models with one enhanced complexity model (Sprite Bottle)
