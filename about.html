About This 3D Application

Project Overview

This 3D Product Showcase application was developed as part of a university assignment to demonstrate the integration of 3D models into a web application using modern web technologies. The application allows users to view and interact with 3D models of beverage products.

Technologies Used
3D Modeling & Rendering
  • Three.js - JavaScript 3D library
  • GLTF/GLB file format for 3D models
  • Blender - 3D modeling software
Web Technologies
  • HTML5 & CSS3
  • JavaScript (ES6+)
  • Bootstrap 5 - CSS framework
Key Features
Interactive 3D Models

Rotate, zoom, and pan to view the 3D models from any angle.

Wireframe Mode

Toggle wireframe mode to see the model's geometric structure.

Lighting Controls

Adjust lighting parameters to view the models under different conditions.

Responsive Design

The application adapts to different screen sizes and devices.

Model Information

The 3D models featured in this application include:

  1. Jar Container - A brown container with ridged cap for easy opening
  2. Soda Can - Bright red aluminum can with pull tab
  3. Thermal Bottle - Yellow insulated bottle with metallic cap and handle

All models were created using Blender and exported in GLB format for web optimization.
