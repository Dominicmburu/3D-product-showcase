Original Work

The following components of this project represent my original work:

  1. 3D Models: All three 3D models (Coke Can, Fanta Can, and Sprite Bottle) were created by me using Blender 3D modeling software specifically for this assignment. The models were designed with attention to detail, appropriate texturing, and optimized for web display. The Sprite Bottle model includes additional complexity in its contoured design for enhanced visualization.
  2. Application Architecture: The structure and organization of the web application, including the HTML layout, CSS styling, and JavaScript functionality, were designed and implemented by me based on the knowledge gained during laboratory sessions but applied to an original design concept.
  3. User Interface: The complete user interface design, including the layout, color scheme, interactive elements, and responsive behavior, represents my original work. I have designed a cohesive and intuitive interface that enhances the 3D viewing experience.
  4. Interactive Controls: All the control mechanisms for manipulating the 3D models, adjusting lighting, toggling wireframe mode, and other interactive features were implemented by me using JavaScript.
